1. Before going into the website, Go to the source code folder, and go to "database" folder, there will be a file named "wbvs.sql"
2. Import that file to phpmyadmin
3. Make sure that the database name in phpmyadmin is "wbvs"
4. Now the website can be used


*Admin Access
To access to admin page, Go to url and type: http://localhost/wbvs/admin/adminlogin.php
You will have to logout first from any logged in user to be able to go to this admin login page.
